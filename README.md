# 💧 Maji Ndogo Water Access Analysis (SQL Project Portfolio)

## 📌 Project Overview
This project is a multi-part SQL-based data analysis case study focused on understanding water access, quality, and infrastructure challenges in Maji Ndogo.

The analysis follows a structured, real-world data workflow, from initial exploration to actionable insights, with an emphasis on data quality, validation, and decision-making.

---

## 🧩 Project Structure

This project is organized into four parts:

1. **Data Exploration (Part 1)**  
   Initial exploration of the dataset, understanding structure, key variables, and identifying potential data quality issues.

2. **Data Cleaning, Transformation & Exploratory Analysis (Part 2)**  
   Cleaning inconsistencies, transforming data, and performing deeper analysis to uncover patterns in water access, population distribution, and queue dynamics.

3. **Data Validation & Investigative Analysis (Part 3)**  
   Validating data accuracy by integrating audit data, identifying inconsistencies, and investigating potential sources of error.

4. **Analysis & Insights (Part 4)**  
   Synthesizing findings into actionable insights and recommendations to support data-driven decision-making.

> ⚠️ This project is currently under development. Subsequent parts (Part 3–4) will be uploaded progressively.

---

## 🎯 Objectives

- Assess water access across different regions and source types  
- Evaluate water quality and reliability  
- Identify inefficiencies such as long queue times  
- Detect inconsistencies and potential data integrity issues  
- Generate insights to support infrastructure and policy decisions  

---

## 🛠️ Tools & Technologies

- **SQL (MySQL)** – Core analysis and data manipulation  
- **Jupyter Notebook** – Documentation and workflow presentation

## 📊 Key Highlights

- Performed SQL-based analysis across multiple related tables  
- Cleaned and transformed data to ensure consistency and usability  
- Validated data using audit comparisons and identified inconsistencies  
- Explored patterns in water access, usage, and queue times  
- Applied analytical reasoning to investigate data quality issues

This project is based on concepts and guided scenarios from the ALX Data Analyst Programme.

Among the datasets provided during the programme, this dataset was selected due to its relevance to real-world challenges related to water access and sustainability (aligned with Sustainable Development Goals).

All SQL queries, analysis, and interpretations presented in this repository were completed independently.
